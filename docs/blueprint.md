# **App Name**: NeuroKids 3D

## Core Features:

- Interactive 3D Worlds: Allow children to explore Maths, Physics, and Astronomy concepts within visually engaging 3D environments.
- Gamified Learning: Introduce mini-games and challenges that reward progress and promote learning retention through game mechanics.
- Parent Dashboard: Provide a dedicated dashboard for parents to track their child's progress, time spent learning, and subject performance.
- Adaptive Learning Tool: Use generative AI to analyze student progress and dynamically adjust the difficulty of games and challenges. The AI will also analyze usage data as a tool to help recommend new learning content and levels.
- Child Profiles: Enable personalized learning experiences with profiles to save progress and preferences.
- Safe Environment: Implement safety features to ensure a secure and child-friendly experience, including content filtering and time limits.
- Rewards and Badges: Award badges and rewards for completing levels and challenges to encourage continuous learning.

## Style Guidelines:

- Primary color: Vibrant purple (#9D4EDD) to evoke imagination and curiosity.
- Background color: Light lavender (#F5EEFE) to create a soft, inviting backdrop.
- Accent color: Electric blue (#4ADEDE) to highlight interactive elements and rewards.
- Body and headline font: 'Space Grotesk' sans-serif, which provides a techy yet readable appearance suitable for all text on screen
- Use 3D icons that are visually appealing and related to the learning topics.
- Design a floating UI with glassmorphism cards to give a modern, immersive feel.
- Incorporate smooth Framer transitions and engaging animations like hover effects and confetti for success moments.
